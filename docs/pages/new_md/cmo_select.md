**select**/mo\_name/
mo\_name is type character, required.

Selects Mesh Object mo\_name to be the Current Mesh Object. If mo\_name
does not exist a new Mesh Object will be created using the Default Mesh
Object as the template.

 
**EXAMPLES:**
**cmo/select**/mo\_tet2
