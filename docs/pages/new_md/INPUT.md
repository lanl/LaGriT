 **INPUT OR INFILE**

  These commands instruct LaGriT to begin processing commands from a
  file. The **infile** commands may be nested. Each  set of commands
  should be terminated with a **finish** command.

 **FORMAT:**

  **infile**/file\_name

  **input**/file\_name
