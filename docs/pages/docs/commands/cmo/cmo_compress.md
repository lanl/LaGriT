**** 

 ompress**/ mo\_name

  ** mo\_name** is type character, default is **-cmo-**

  Shortens all memory managed arrays for Mesh Object mo\_name to their
  actual lengths.

 

**EXAMPLES:**

mo/compress**/ mo\_tet2

mo/compress/-cmo-**

mo/compress**

mo/compress/-all-**
